Геомодели для оптимизации работы в QGIS

Для оптимизации скорости подготовки данных были разработаны геомодели:

1. Объединение слоев по зонам МСК

Модель по объединению множества слоев, которые находятся в 3 зонах МСК, и приведению их в одну выбранную зону МСК

Смотреть демонстрацию

2. Создание точек из Excel таблиц

Модель по созданию точек из множества Excel таблиц координат Береговых линий (БЛ), Водоохранных зон (ВОЗ), Прибрежных защитных полос (ПЗП)

3. Отсечение координатных точек

Модель по отсечению координатных точек БЛ из ВОЗ и ПЗП, и выборка номеров точек для надписывания нумерации

4. Расчет стыков разграфки

Модель по расчитыванию смежных полей стыков разграфки по аналогии инструмента из ArcGIS

Смотреть демонстрацию
